\begin{itemize}

Data
\begin{itemize}

\end{itemize}

Data
\begin{itemize}
\item 227,835 radiology reports (MIMIC-CXR-RoCo)
\item Chest X-ray images with bounding box annotations for visual grounding
\item Comprehensive clinical metadata including temporal context
\end{itemize}


Contributions
